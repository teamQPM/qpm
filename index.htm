English:

QPM (QAC based Project Manager) is an Project Manager for Harbour (or xHarbour) and MiniGui (created by Roberto Lopez) with Incremental compilation for all MiniGui versions, including Oficial MiniGui 3.x and MinGw C Compiler.  Also it allows to create Console applications and libraries (LIB and A) for Borland (MiniGUI 1.x) and for MinGW (MiniGUI 2.x)

Español:

QPM (QAC Based Project Manager) es un Manejador de proyectos que sirve para Compilar programas Harbour y xHarbour con la librería gráfica MiniGui creada por Roberto López.

Lo interesante de este utilitario es que permite compilar en forma incremental con todas las versiones de MiniGui del mercado y sus respectivos compiladores C, inclusive MiniGui 2.x con el compilador C MinGW.
