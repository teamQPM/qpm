QPM (QAC Based Project Manager)

 

July 13, 2009 Last Release                     13 de Julio de 2009: Ultima Versión

English:

QPM (I want something comfortable !!!) is an Project Manager for Harbour (or xHarbour) and MiniGui (created by Roberto Lopez) with Incremental compilation for all MiniGui versions, including Oficial MiniGui 2.x and MinGw C Compiler.  Also it allows to create Console aplications and libraries (LIB and A) for Borland (MiniGUI 1.x) and for MinGW (MiniGUI 2.x)

Español:

QPM (QAC Based Project Manager) es un Manejador de proyectos que sirve para Compilar programas Harbour y xHarbour con la librería gráfica MiniGui creada por Roberto López.

Lo interesante de este utilitario es que permite compilar en forma incremental con todas las versiones de MiniGui del mercado y sus respectivos compiladores C, inclusive MiniGui 2.x con el compilador C MinGW.

 

Free QPM Download:  QPM_04_11_0016_Install.exe 

 

Recomended for:

 

Oficial MiniGui 1x Build 108 (with Borland, Harbour and xHarbour 0.99.50)

Extended MiniGui 1.6 Build 66  (with Borland, Harbour and xHarbour 1.2.01 )

Oficial MiniGui 2.8.8 (with MinGW, Harbour and xHarbour 1.2.01 )

OOHG (Object Oriented Harbour GUI) 1.x of 2009/03/27 (March Version) (with Borland, MinGW, Pelles, Harbour and xHarbour 1.2.01 )
