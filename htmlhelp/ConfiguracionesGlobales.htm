¿Cómo se configuran los parámetros o seteos globales?
    

QPM tiene una configuración Global en donde se especifican los nombres de Directorios de las diferentes versiones de MiniGui, el nombre del Editor a utilizar y una serie de parámetros más referidos a las tools que provee o da soporte.

Esta configuración es accesible desde la opción Global Options del menú Settings:



Configurando el Editor de Texto y las herramientas de manejo de Forms:

Editor:

Program Editor for Text Files: Ubicación del editor de texto que se usará para editar los programas fuentes y/o cualquier otro texto que sea necesario

Use Long Name: Por default, para evitar problemas con los nombres de directorios largos o con caracteres en blanco, los parámetros pasados al editor, como ser el camino y nombre del fuente a editar, se le proporcionan en formato nombre de archivo de DOS 8.3. Activando esta opción, el nombre será proporcionado tal como es. Notese que varios productos del mercado tienen problemas en manejar este tipo de nombres de Directorios y Archivos, por lo que se recomienda no activar esta opcion salvo que sea necesaria.



Suspend Control Edit: Este seteo afecta tanto a Sources como a Headers y Forms. En el caso de Forms, sólo tiene efecto cuando se los edita con el Editor predeterminado, es decir, cuando NO se utilizan las tools de edición como ser HMI+ o HMGS-Ide.
Tilde esta opción si desea que QPM no efectue control sobre los archivos que se estan editando. Esta opción se agregó a QPM porque hay programas de edición de texto que no soportaban la facilidad de control de edición de QPM. Debe tildarse en caso de problemas con su editor de texto, sino, se recomienda no tildar, ya que a futuro se prevee la facilidad de Quick Edit, o sea, editar el fuente con el propio QPM y si esta opcion está tildada, dicha facilidad no estará disponible.
Si esta opción está tildada no se mostrará la letra E cuando se esté editando un fuente.

Nota: La activación de esta opción desactiva el Forzado de Recompilaciones Automaticamnete (Ver ¿Como forzar automaticamente la recompilacion de modulos?)



Form Tool Locate:

HMI+: Ubicación de la herramienta de modificación de Forms (Pantallas) creada por Ciro Vargas Clemow. Se requiere una versión 1.66 o superior.

HMGSIDE: Ubicación de la herramienta de modificación de Forms (Pantallas) creada por Walter Formigoni. Se requiere una versión 0.0.7 o superior.

DBF Tool Locate: QPM provee una tool para el manejo de tablas DBF: DBFView, de Grigory Filatov

Pero además, ofrece la posibilidad de utilizar alguna otra herramienta para el acceso a las tablas DBF, esta herramienta externa deberá configurarse en esta pestaña.

Select DBF Tool: Ubicación de la herramienta de modificación de DBF que el usuario desea utilizar



Do Not Use Quotes in DBF name: Esta opción se utiliza para que al llamar a la herramienta externa de manejo de DBF no se le agreguen las comillas a los extremos del nombre de la DBF que intentamos abrir. Esto sólo se usa en caso de que la herramienta externa lo requiera.
Por ejemplo: si usamos c:\tool\DBAlite.exe como herramienta y queremos abrir la base c:\DBFs\Test.dbf dependiendiendo de esta opcion, el comando interno que ejecuta QPM será:
Run( c:\tool\DBAlite.exe "c:\DBFs\Test.dbf" ) si el control es FALSE y
Run( c:\tool\DBAlite.exe c:\DBFs\Test.dbf ) si el control es TRUE




FOLDERS

Nota: Dentro de los folders para MiniGui sólo se deberán completar los de la librería MiniGui que deseamos utilizar. Y en cada caso, sólo es necesario especificar un compilador de la familia Harbour, es decir, si vamos a utilizar MiniGui con Harbour no es necesario especificar el folder de xHarbour y viceversa. Sin embargo, se pueden completar todos los Folder aunque se trate de MiniGui's o [x]Harbour que no utilizaremos

Folders for Oficial MiniGui 1.x and Borland C: Completar esta configuración si se desea utilizar Oficial MiniGui versión 1.x

MiniGui Folder: Directorio principal de la minigui (Default C:\MiniGui1x)

Borland C Folder: Directorio donde reside el Compilador C (Default C:\Borland\Bcc55)

Harbour Folder: Directorio donde reside el Compilador Harbour (Default C:\MiniGui1x\Harbour) (Opcional si se utilizará xHarbour)

xHarbour Folder: Directorio donde reside el Compilador xHarbour (Default C:\MiniGui1x\xHarbour) (Opcional si se utilizará Harbour)



Folders for Oficial MiniGui 3.x and MinGW: Completar esta configuración si se desea utilizar Oficial MiniGui versión 3.x

MiniGui Folder: Directorio principal de la minigui (Default C:\HMG)

MinGW C Folder: Directorio donde reside el Compilador C (Default C:\HMG\MinGW)

Harbour Folder: Directorio donde reside el Compilador Harbour (Default C:\HMG\Harbour) (Opcional si se utilizará xHarbour)

xHarbour Folder: Directorio donde reside el Compilador xHarbour (Default C:\HMG\xHarbour) (Opcional si se utilizará Harbour)



Folders for Extended MiniGui 1.x and Borland C: Completar esta configuración si se desea utilizar Extended MiniGui versión 1.x

MiniGui Folder: Directorio principal de la minigui (Default C:\MiniGui)

Borland C Folder: Directorio donde reside el Compilador C (Default C:\Borland\Bcc55)

Harbour Folder: Directorio donde reside el Compilador Harbour (Default C:\MiniGui\Harbour) (Opcional si se utilizará xHarbour)

xHarbour Folder: Directorio donde reside el Compilador xHarbour (Default C:\MiniGui\xHarbour) (Opcional si se utilizará Harbour)



Folders for OOHG 3.x and Borland C: Completar esta configuración si se desea utilizar Object Oriented Harbour Gui 3.x y el compilador Borland C

MiniGui Folder: Directorio principal de la minigui (Default C:\OOHGforBorland)

Borland C Folder: Directorio donde reside el Compilador C (Default C:\Borland\Bcc55)

Harbour Folder: Directorio donde reside el Compilador Harbour (Default C:\OOHGforBorland\Harbour) (Opcional si se utilizará xHarbour)

xHarbour Folder: Directorio donde reside el Compilador xHarbour (Default C:\OOHGforBorland\xHarbour) (Opcional si se utilizará Harbour)



Folders for OOHG 3.x and MinGW: Completar esta configuración si se desea utilizar Object Oriented Harbour Gui 3.x y el compilador MinGW

MiniGui Folder: Directorio principal de la minigui (Default C:\OOHGforMinGW)

MinGW C Folder: Directorio donde reside el Compilador C (Default C:\MinGW)

Harbour Folder: Directorio donde reside el Compilador Harbour (Default C:\OOHGforMinGW\Harbour) (Opcional si se utilizará xHarbour)

xHarbour Folder: Directorio donde reside el Compilador xHarbour (Default C:\OOHGforMinGW\xHarbour) (Opcional si se utilizará Harbour)



Folders for OOHG 3.x and Pelles C: Completar esta configuración si se desea utilizar Object Oriented Harbour Gui 3.x y el compilador Pelles C

MiniGui Folder: Directorio principal de la minigui (Default C:\OOHGforPelles)

Pelles C Folder: Directorio donde reside el Compilador C (Default C:\Pelles)

Harbour Folder: Directorio donde reside el Compilador Harbour (Default C:\OOHGforPelles\Harbour) (Opcional si se utilizará xHarbour)

xHarbour Folder: Directorio donde reside el Compilador xHarbour (Default C:\OOHGforPelles\xHarbour) (Opcional si se utilizará Harbour)
