                             Welcome to QPM
                             ==============

QPM (QAC based Project Manager) is a project manager for software applications
based on Harbour and xHarbour languages, it's contributions, specialy the
graphic support libraries for Windows: Harbour MiniGUI, Harbour MiniGUI Extend
and Object Oriented Harbour GUI (OOHG), and Borland, Pelles and MinGW C/C++
compilers.

QPM is the Open Source continuation of the application QAC created by
CarozoDeQuilmes (inspired on the MPM utility designed by Roberto López, creator
of Harbour MiniGUI).

Assists programmers in the processes of program developing, compilation and
linkedition, providing tools (interactive and batch) for:

+ Incremental compilation of GUI, console and mixed type programs.
+ Building static libraries and DLL access interfaces.
+ Building CHM and HTML help files.
+ Visualizing Sources, Forms, Headers and DBFs.
+ Searching into Sources, Forms, Headers and DBFs.
+ Displaying the names of functions included in static and dynamic libraries.
+ Accessing external utilities to edit Forms and DBFs.
+ Executing aditional processes after build.
+ Compressing EXEs with UPX utility.
+ Visualizing error files resulting from compilation and execution.
+ Visualizing .PPO files resulting from preprocessor.

The QPM web site is at <URL:http://sourceforge.net/p/qpm/>. If you have any
problem with this copy of QPM, please visit our web site and ensure that you are
using the latest release.

If you have any questions about QPM, please subscribe to the users list at
<URL:https://lists.sourceforge.net/lists/listinfo/qpm-users/> and ask there.
Also, please be sure to read the QPM's help file (from QPM menu or from this
site: http://qpm.sourceforge.net) and the documentation that comes with QPM
(start with ./dirstruc.txt because is the map to the QPM files).


                             Bienvenido a QPM
                             ================

QPM (QAC based Project Manager) es un administrador de proyectos para
aplicaciones de software basadas en los lenguajes de programación Harbour y
xHarbour, sus contribuciones, en especial las librerías de soporte gráfico para
Windows: Harbour MiniGUI, Harbour MiniGUI Extend y Object Oriented Harbour GUI
(OOHG), y los compiladores C/C++ Borland, Pelles y MinGW.

QPM es la continuación Open Source de la aplicación QAC creada por
CarozoDeQuilmes (inspirado en el utilitario MPM diseñado por Roberto López,
creador de Harbour MiniGUI).

Asiste al programador en los procesos de desarrollo, compilación y ensamblado de
programas, al proveer herramientas (interactivas y batch) para:

+ Compilación incremental de programas en modo GUI, consola y mixtos.
+ Crear librerías e interfaces de acceso a DLLs.
+ Generar archivos de ayuda en formato CHM y HTML.
+ Visualizar Sources, Forms, Headers y DBFs.
+ Realizar búsquedas en Sources, Forms, Headers y DBFs.
+ Listar las funciones incluidas en librerías estáticas y dinámicas.
+ Acceder a utilitarios externos para la edición de Forms y DBFs.
+ Ejecutar procesos adicionales luego de finalizado el ensamblado.
+ Comprimir el ejecutable utilizando el utilitario UPX.
+ Visualizar arhivos de errores resultantes de la compilación y ejecución.
+ Visualizar archivos .PPO resultantes de la ejecución del preprocesador.

El sitio web de QPM está ubicado en <URL:http://sourceforge.net/p/qpm/>. Si
